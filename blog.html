Var
1. var is function scoped when it is declared within a function. This means that it is available and can be accessed only within that function.
2.var variables can be re-declared and updated
3.Hoisting of var
So var variables are hoisted to the top of their scope and initialized with a value of undefined.

Let

1.let is block scoped
2.let can be updated but not re-declared.
3.Hoisting of let

Const

1. const declarations are block scoped
2. const cannot be updated or re-declared

differences between regular and array function

1.Implicit Return
In regular function, you have to use return keyword to return any value. If you don’t return anything then the function will return undefined. and Arrow functions behave in the same way when returning values. If the arrow function contains one expression, you can omit the curly braces, and then the expression will be implicitly returned.
2️. Arguments binding
In regular function, Arguments keywords can be used to access the arguments of which passed to function.Arrow functions do not have an arguments binding.
3.new
Regular functions are constructible, they can be called using the new keyword.arrow functions can never be used as constructor functions. Hence, they can never be invoked with the new keyword
